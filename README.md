# Lab Commands Reference

| Command / Parameter | Description |
| :--- | :--- |
| `git pull` | Pull latest version of a file |
| `git stage <Filename>` | Select specific file to update |
| `git commit -m <Message>` | Save changes to local repository history |
| `git push` | Make changes public |
| `(ip/if)config` | List device network properties |
| `ps -e` | List all running processes |
| `ssh` | Initiate login to another machine via SSH |
| `apt update` | Refresh system packages list |
| `apt upgrade` | Download and install updates for all existing software |
| `apt install apache2` | Install Apache2 web server |
| `ufw enable` | Enable firewall |
| `ufw allow 80/tcp` | Open Port 80 (TCP) for HTTP traffic |
| `ls -l` | List files & folders in detailed format |
| `find <path> <parameter>` | Find files and directories |
| `-type f` | Find file *(find parameter)* |
| `-type d` | Find directory *(find parameter)* |
| `-name <name>` | Indicate the target name *(find parameter)* |
| `grep <parameter> "target" <path>`| Find text pattern and output results |
| `-R` or `-r` | Recursive search *(grep parameter)* |
| `#!/bin/bash` | Bash shebang (defines the script interpreter) |
| `cp` | Create a copy of a file or directory |
| `zip` | Compress multiple files into one `.zip` file |
| `cd` | Change directory |
| `scp` | Secure file transfer using SSH |
| `date` | Print or assign the current date/time |
| `nslookup` | Look up DNS to find domain name or IP address |
| `certbot --apache` | Create or configure SSL Certificate for Apache |
| `rsync` | Copy and synchronize files between locations |
